# Statement of Work: NPP & Osko Implementation

**Client:** Heritage Mutual Bank  
**Date:** March 15, 2023  
**SOW Reference:** SOW-2023-001  
**Status:** Executed  

## 1. Executive Summary

This Statement of Work (SOW) outlines the implementation of Cuspac's Real-Time Payments (NPP) solution for Heritage Mutual Bank ("Client"). The project enabled Heritage to offer instant Osko payments and PayID services to its 150,000 members, ensuring competitive parity with major banks.

## 2. Scope of Work

The scope included establishing NPP connectivity, integrating with the core banking system (Ultradata), and certification with NPP Australia (NPPA).

### In Scope
- Provisioning of Cuspac NPP Gateway (Test & Prod)
- Implementation of ISO 20022 message flows (nct, pacs.008, pacs.002)
- PayID API integration
- Dispute Management Workflow setup
- NPPA Industry Test certification support

### Out of Scope
- Mobile App UI changes (Vendor provided API only)
- Marketing campaigns for Osko launch
