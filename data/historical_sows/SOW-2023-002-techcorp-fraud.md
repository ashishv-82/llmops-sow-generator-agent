# Statement of Work: PayTo Mandate Management

**Client:** SwiftPay Fintech  
**Date:** April 10, 2024  
**SOW Reference:** SOW-2024-002  
**Status:** Executed  

## 1. Executive Summary

SwiftPay Fintech ("Client") engaged Cuspac to implement PayTo capabilities for their B2B recurring billing platform. This project delivered the ability to create, amend, and initiate payments against PayTo mandates via Cuspac's API.

## 2. Scope of Work

The project delivered a RESTful API integration for Mandate Management Service (MMS) and payment initiation.

### In Scope
- PayTo Mandate Creation APIs
- PayTo Payment Initiation APIs
- Webhook setup for Mandate Status changes (Active, Suspended, Cancelled)
- Sandbox environment for developer testing

### Out of Scope
- End-user UI for mandate approval (Banking app scope)
- Legacy Direct Debit file conversion
